**CommandUtils**

CommandUtils is a set of utility commands for laravel and lumen
to replace or add back commands.

**Provided Commands**

| Command  | Laravel/Lumen Versions |
| -------  | ---------------------- |
| app:name | ``laravel 6.x, lumen 5.8 +`` |
